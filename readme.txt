WordPress plugin to generate vCard from Contact Form 7 inquiries

== Description ==

Contact Form vCard Generator WordPress plugin allows you to generate vCard for all contact inquiries. Contact Form 7 is more like install, activate, configure and ready to go plugin, hence it is widely used across the WordPress sites. We often need to import/export contacts here and there, vCard is one of the way to do that. This plugin is really helpful when you like to generate vCard from Contact inquires generated via Contact Form 7.

= Overview =
- Option to use the Contact Form 7 using Shortcode.
- Map Contact Form 7 fields with vCard fields.
- Attach vCard in Contact Form 7 email notification upon form submission.
- Download vCard for individual form entry.
- Download all form entries in single vCard file.

For detailed information on every features of this plugin including how to map contact form 7 values with vCard attributes you can visit this article [WordPress Contact Form vCard Generator](https://freelancer-coder.com/wordpress-contact-form-vcard-generator)

If you have any suggestions/comments or you like to get further information on this plugin please feel free to [connect with me](https://freelancer-coder.com/contact-wordpress-developer/). I will make sure to provide guidelines or some useful information for the addressed situation.

== Installation ==

= Prerequisites =
1. [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) plugin is necessary to use Contact Form vCard Generator.
2. If you are looking to use any existing contact form (already built with Contact Form 7) then you must edit and save that contact form.

Installation process is very simple for Contact Form vCard Generator plugin. Ways to install plugin:

= Installation with FTP: =

      1. Download Contact Form vCard Generator plugin.
      2. Extract plugin.
      2. Upload Contact Form vCard Generator directory to the '/wp-content/plugins/' directory.
      3. Go to Plugins option from left menu and activate 'Contact Form vCard Generator' plugin from the list.
      
= Installation with Upload method via WordPress admin panel: =

      1. Download Contact Form vCard Generator plugin.
      2. Go to plugins page by clicking on Plugins menu item from left menu.
      3. Click on 'Add New' option.
      4. Upload the plugin and activate.

= Support available? =
Yes, I will provide support for any of the query, please allow 48 hours to get back to you.

= How many contact forms can be configured? =
At a time this plugin will work with only one contact form. In case if you like to generate vCards from multiple contact forms then you may need to use shortcode of every contact form one by one. Please be sure that once you change the shortcode of contact form and submit, you'll need to map contact form fields with vCard fields again.

= Can this plugin generate hCard? =
No, this plugin only supports vCard.

= Will it work on my Theme? =
This plugin is only for backend.
